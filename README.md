# qBitController

Простое iOS-приложение (SwiftUI) для управления qBittorrent через его Web API v2.

## Возможности
- Список торрентов с прогрессом, скоростями загрузки/отдачи и состоянием
- Добавление по magnet-ссылке или URL .torrent-файла
- Старт / пауза торрента (кнопка в строке или свайп)
- Удаление (свайп влево)
- Автообновление списка каждые 3 секунды + pull-to-refresh
- Настройки подключения с проверкой соединения

## Запуск
1. Откройте `qBitController.xcodeproj` в Xcode.
2. Выберите симулятор или устройство и нажмите Run (⌘R).
3. При первом запуске откройте настройки (шестерёнка) и укажите:
   - **Сервер** — адрес Web UI, например `http://192.168.0.2:8080`
   - **Логин** и **Пароль** от Web UI qBittorrent

Web UI включается в qBittorrent: Настройки → Web UI.

## Структура
- `QBittorrentClient.swift` — клиент Web API (авторизация по cookie, запросы)
- `Models.swift` — модель торрента и форматирование
- `ContentView.swift` — список и тулбар
- `TorrentRowView.swift` — строка торрента
- `AddTorrentView.swift` — добавление
- `SettingsView.swift` — настройки подключения

## Совместимость
Поддерживаются и новые (`/start`, `/stop`), и старые (`/resume`, `/pause`)
имена эндпоинтов qBittorrent. Минимальная версия iOS — 17.0.

HTTP-соединения разрешены (`NSAllowsArbitraryLoads`), так как qBittorrent
в локальной сети обычно работает без TLS.
